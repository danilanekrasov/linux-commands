bash
| Команда | Описание |
| ------- | --------- |
| #!/bin/bash    | путь до интерпретатора
| echo "текст"    | вывести текст

| #    | комментарий
| $<имя> или ${<имя>}    | обращение к переменной
| $1 $2 ... $n    | обращение к аргументам скрипта
| $0    | имя скрипта
| $#    | количество аргументов
